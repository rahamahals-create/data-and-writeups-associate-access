# Data & Writeups Assistant Portal

This repository now includes a lightweight, accessible starter website to support employees working on:

- data updating projects
- writeup generation and status communication
- time-saving strategy planning

## Files

- `index.html` - semantic page structure with accessible landmarks and form labels
- `styles.css` - readable, keyboard-friendly styles
- `script.js` - quick actions and strategy generation behavior

## Run locally

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.
